如何複製 WebStorm 專案 ?

直接 Hack .idea 目錄
Contents
  1. Version
  2. Symptons
  3. Recipes
  4. Conclusion

有時想以一 Project 為基底，將該專案複製後再重新命名，這樣雖然可行，但在 WebStorm 的 Project Explorer 總算顯示原本 Project 名稱，非常礙眼。

Version

macOS Mojave 10.14
WebStorm 2018.2.5

Symptons

  1. 雖然已經將目錄名稱改成 vue-todo，但右側仍然顯示原本的目錄的 vue-todo-lis-api，非常礙眼

Recipes

  1. 在專案目錄下的 .idea 目錄下，這是個隱藏目錄
  2. 找到原本專案名稱的 .iml 檔案

  1. 一樣在 .idea 目錄下
  2. 將 .iml 檔名改成新的目錄名稱

  1. 選擇 .idea 目錄下的 modules.xml
  2. 以 Sublime Text 開啟

  • 將兩個 .iml 改成新的檔案名稱

  1. 重新用 WebStorm 開啟專案，一切就正常了

Conclusion

  • 這並不是正規做法，算是 hack WebStorm 了，不過實務上卻非常好用
