現代前端的 Tooling 繁雜， 一直是跨入前端的門檻，讓 CLI 成為 Tooling 的抽象層，由 CLI 幫我們解決繁瑣的 Tooling 整合。

由於 Vue CLI 3 的發表，Vue 的 CLI 又向前邁進一大步，本文介紹如何使用 Vue CLI 3 快速建立 Vue 專案。

Version

macOS High Sierra 10.13.6
Node.js 8.11.3
Yarn 1.9.4
Vue.js 2.5.17
Vue CLI 3.0.0

Node.js

現代前端都基於 Node.js 生態系，Vue 也不例外，首先必須安裝 Node.js。

Vue CLI 3 官網推薦的版本是 Node.js 8.11.0 以上。

Node.js 版本進步很快，又不像 .NET Core 可以同時並存多個版本，比較不建議將單一 Node.js 版本安裝在系統，而是另外使用 Node.js 版本管理工具作為抽象層，或使用 Docker，將來方便管理 Node.js。

  • Mac / Linux : nvm
  • Windows : nvm-windows

使用 nvm 安裝 Node.js 指定版本。

Yarn

Node.js 內建 NPM 套件管理工具，但 NPM 的演算法實在不優，速度很慢，Facebook 另外提出了 Yarn 管理前端套件。

使用 NPM 以 g lobal 方式安裝 Yarn。

將來安裝前端套件，將以 Yarn 取代 NPM。

Vue CLI

使用 Yarn 以 g lobal 方式安裝 Vue CLI。

建立 Project

  1. 選擇 project 類型，先按 ↩ 選擇預設值即可

開始建立專案。

執行 Vue

使用 yarn serve 執行 Vue。

  1. 進入 hello-world 目錄
  2. 輸入 yarn serve 執行 Vue

  1. Vue 啟動在 http://localhost:8080

Vue 順利執行在 localhost:8080。

Conclusion

  • Vue CLI 3 讓我們類似 Angular CLI 快速建立專案，也將相關的設定 tooling 都準備好，大幅降低現代前端的進入門檻
