Vue CLI 3 之 Instant Prototyping

不用建立 Vue 專案就可 demo
Contents
  1. Version
  2. 安裝 CLI-Service Global
  3. 建立 .vue
  4. 執行 Vue
  5. Conclusion
  6. Reference

Vue CLI 3 提供了一個很酷的 Instant Prototyping，在學習 Vue 時非常有用，讓我們不用建立 Vue 專案，只要一個 .vue 檔就可開始練習與 demo。

Version

macOS High Sierra 10.13.6
Node.js 8.11.3
Yarn 1.9.4
Vue 2.5.17
Vue CLI 3.0.0
WebStorm 2018.2.1

安裝 CLI-Service Global

Vue CLI 並沒有內建 CLI-Service Global，要另外安裝。

建立 .vue

用任何工具建立 .vue 皆可，如存檔命名為 hello.vue。

  1. 就算使用 Vim 建立 .vue 也行

執行 Vue

vue serve 後面接 檔案名稱，就可執行 Vue。

Vue 啟動在 http://localhost:8080/

Vue 順利執行在 localhost:8080。

Conclusion

  • Vue CLI 3 的 Instant Prototyping 讓我們在學習測試 Vue 時更方便，不用再使用 Yarn 建立新專案，只要一個 .vue 檔即可
