如何使用 Vue 搭配 ESLint + Airbnb Style Guide ?

Vue CLI 3 已經整合 ESLint 與著名的 Airbnb JavaScript Style Guide，只要跟著 Vue CLI 3 的 Wizard，我們也能輕易地將 ESLint 與 Airbnb 整進 Vue 專案。

Version

macOS High Sierra 10.13.6
Node.js 8.11.3
Yarn 1.9.4
Vue.js 2.5.17
Vue CLI 3.0.0
WebStorm 2018.2.1

建立 Project

選擇 Feature

  1. 不選擇 babel, eslint，選擇 Manually select features

選擇 Babel 與 Linter

  1. 選擇 Babel
  2. 選擇 Linter / Formatter

  1. 選擇 ESLint + Airbnb config

存檔時執行 ESLint

  1. 選擇 Lint on save

Config 存檔方式

  1. 選擇 In dedicated config files，也就是設定會單獨在 .eslintrc.js，而不是存到 package.json 內

將設定存成預設值

  1. 選擇 Y 將 ESLint + Airbnb 存成預設值
  2. 將預設值存成 .vuerc

.vuerc 將存到 home directory 根目錄下，將來 vue create-project 時，將可選擇 .vuerc 內的設定

將來建立專案時，將多了 .vuerc (babel, eslint) 選項，也就是剛剛存的預設值。

選擇套件管理工具

  1. 選擇 Yarn，將以 Yarn 作為標準的套件管理工具

Vue CLI 建立專案成功。

WebStorm

  • WebStorm 能正確搭配 ESList 與 Airbnb Style Guide 發出警告

Conclusion

  • Vue CLI 3 已經將 ESLint 與 Airbnb Style Guide 整好，WebStorm 也能聰明地抓到 Vue CLI 的設定進行檢查

Reference

Airbnb, Airbnb JavaScript Style Guide
Vue.js, Vue CLI 3
