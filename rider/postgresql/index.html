如何使用 Rider 連接 PostgreSQL ?

直接在 Rider 內連接到 PostgreSQL
Contents
  1. Version
  2. Rider
  3. Conclusion

Rider 內建資料庫連線工具，可以直接連上任何資料庫，當然也包含 PostgreSQL。

Version

macOS High Sierra 10.13.3
Docker for Mac 18.03.1-ce-mac60 (24312)
PostgreSQL 10.3
Rider 2018.1

Rider

View -> Tool Windows -> Database

右側顯示 Database View。

Rider 連進 PostgreSQL。

  1. 按 +，選擇 Data Source
  2. 選擇 PostgreSQL

Rider 支援 PostgreSQL，不過預設並沒有安裝 PostgreSQL 驅動程式。

  1. 按 Download 下載 PostgreSQL 驅動程式。

  1. User : admin
  2. Password : 12345
  3. 按 Test Connection 測試連接，顯示 Successful

Rider 預設不會顯示所有 table。

  1. 選 Shemas...
  2. 勾選 All schemas
  3. 按 OK 繼續

  1. 展開 tables，選擇 sql_features table
  2. 按 Data Editor 顯示資料

如此 Rider 已經正確連上 PostgreSQL。

Conclusion

  • Rider 已經自帶資料庫連線工具，也包含各資料庫的 driver，可輕鬆連上各種資料庫
