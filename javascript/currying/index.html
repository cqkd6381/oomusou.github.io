深入淺出 FP 之 Currying

徹底搞懂為什麼要使用 Currying
Contents

不只 OOP 有 Design Pattern，事實上 FP 也有不少 Pattern，而 Currying 算是 FP 最基本、且用的最多的 Pattern。

一些正統 FP 語言，如 Haskell、F# … 都在語言內直接支援 Currying；JavaScript 雖然沒有直接支援 Currying，但因為 JavaScript 支援 First-class Function 與 Closure，使得 Currying 在 JavaScript 中使用成為可能。
